# Garage Scheduling

## Dependencies

There are no specific dependencies. The code has no external library dependencies — it uses only the standard C++ 17 library. 
This is the only dependency, have C++17 or later version of g++. 

## Input

The input file is hardcoded as `input.txt` inside the source code itself — no filename needs to be passed via the command line.

To use a different input, simply replace the contents of `input.txt` with your new input before running.

## Test case generation
We have made out own test case, no specific code for generating test cases.

## How to Run

**Compile the program:**
```bash
g++ garage_scheduling.cpp -o garage_scheduling
```

**Run the program:**
```bash
./garage_scheduling
```
